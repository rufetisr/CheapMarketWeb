## ‚ú® Overview
**CheapMarket** is a powerful web application designed to simplify the online shopping experience. Instead of opening dozens of tabs to find the best price, CheapMarket aggregates data from various retailers into a single, clean dashboard. 

Built with performance and user experience in mind, it allows users to search for products and see the best available deals in real-time.

Built with **Google Gemini AI** and **OCR technology**, it empowers users to not only find the lowest prices but also decode complex food labels and track long-term market fluctuations to maximize their savings.

## üöÄ Key Features

### üí∞ Smart Price Comparison
- **Unified Search:** Search across multiple marketplaces simultaneously from a single dashboard.
- **Real-time Data:** Fetches the latest pricing and availability directly from the source.
- **Clean UI/UX:** A minimalist interface focused on speed, clarity, and mobile responsiveness.

### üìì Discount Journals ([/journals](https://cheapmarketweb.onrender.com/journals))
- **Strategic Tracking:** Maintain a personal log of price drops and "best finds" to identify recurring sales cycles.
- **Market History:** Build a database of historical pricing to distinguish between genuine deals and marketing gimmicks.
- **Savings Log:** Keep notes on product quality vs. price to improve future purchasing decisions.

### ü•ó Nutri Analyzer ([/nutrition-analyzer](https://cheapmarketweb.onrender.com/nutrition-analyzer))
- **AI-Powered OCR:** Upload a photo of product nutrition label. The system extracts text automatically using Optical Character Recognition.
- **Gemini AI Insights:** Processes extracted data to provide a "Health Score," identifies hidden additives, and flags high levels of sugar or sodium in simple, actionable terms.
